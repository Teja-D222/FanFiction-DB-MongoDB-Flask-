# FanFiction-DB-MongoDB-Flask-
A functional, end-to-end fan-fiction platform built with MongoDB and Flask

## App Title: The Bulls Fiction Factory
A Flask + MongoDB Fan-Fiction Platform

## Overview
The Bulls Fiction Factory is a lightweight, full-stack web application that allows users to create, view, search, and manage fan fiction stories based on movies. Built using Flask, MongoDB, and a clean HTML/CSS/JavaScript frontend, the platform demonstrates practical implementation of NoSQL databases, routing, CRUD operations, and UI-driven data submission.
This system is ideal for showcasing practical skills in advanced database management, NoSQL modeling, and web application development.

## Application Architecture
The app follows a simple, clean request-flow:
User submits a story via the HTML form
Data is routed to Flask (page 6)
Flask processes and stores the data into MongoDB
The updated content is retrieved and displayed on the homepage
Users can also search or delete entries as needed (pages 9‚Äì11)
A diagram summarizing this flow is provided on page 6, showing how MongoDB, Flask, and HTML interact.
